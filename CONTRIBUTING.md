# Contributing to Universal Academic Advisor

First of all, thank you for wanting to contribute! 🎉
This project is beginner friendly — you do not need to know AI or RAG to contribute.

---

## Easiest way to contribute — Add your college's data

No coding needed. Just add your college's course data.

**Step 1** — Fork this repo (click the Fork button on top right of this page)

**Step 2** — Clone your fork
```bash
git clone https://github.com/YourUsername/universal-academic-advisor.git
cd universal-academic-advisor
```

**Step 3** — Create a folder for your college inside `Data/`

Data/
└── YourCollegeName/
├── courses.json
└── career.json

**Step 4** — Add your data following the format in [Data/DATA_FORMAT.md](Data/DATA_FORMAT.md)

**Step 5** — Push and open a Pull Request with title:
`Add course data for [Your College Name]`

---

## Other ways to contribute

- **Fix a bug** — check the Issues tab for bugs
- **Improve the README** — fix typos, add clearer explanations
- **Suggest a feature** — open an issue and describe your idea
- **Improve the RAG pipeline** — if you know LangChain or FAISS

---

## First time using Git and GitHub?

That is completely fine. Open an issue saying you want to contribute and we will guide you through it step by step.

---

## Code of Conduct

Never giving up is the only rule here. 🔥

Stuck on something? Try again.
PR got rejected? Try again.
Don't understand the codebase? Try again.

This project was built by a student who was also figuring things out.
Nobody here knows everything — we just don't stop trying.

Be kind, be patient, and keep going.
Every expert was once a beginner who refused to quit. 🚀
